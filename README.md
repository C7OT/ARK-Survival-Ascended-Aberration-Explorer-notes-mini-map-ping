# ARK-Survival-Ascended-Aberration-Explorer-notes-mini-map-ping
places markers for every explorer note for Aberration on your mini map

How to install? 
1. open steam and browse local files navigate to C:\Program Files (x86)\Steam\steamapps\common\ARK Survival Ascended\ShooterGame\Saved\Config\Windows\GameUserSettings.txt
2. open the GameUserSettings.txt file and paste the code under [/Script/ShooterGame.ShooterGameUserSettings]
3. save the file.
4. set the GameUserSettings.txt file to read only and apply
5. start your game and enjoy!
